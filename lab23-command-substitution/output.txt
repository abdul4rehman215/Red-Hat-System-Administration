Task 1: Basic Command Substitution

Command:

Backtick Syntax Example
-----------------------

Command:
echo "Today is `date`"

Output:
Today is Sat Feb 22 10:48:32 UTC 2026


Nested Substitution Comparison
------------------------------

Command:
echo "Test 1: $(echo $(date))"

Output:
Test 1: Sat Feb 22 10:48:45 UTC 2026


Command:
echo "Test 2: `echo \`date\``"

Output:
Test 2: Sat Feb 22 10:48:51 UTC 2026
