Task 2: Inline Text Replacement with sed
--------------------------------------------------

-bash-4.2$ echo -e "RHEL 8\nRHEL 9\nFedora 38" > versions.txt

-bash-4.2$ cat versions.txt

RHEL 9
Fedora 38


-bash-4.2$ sed -i 's/RHEL/Red Hat/g' versions.txt

-bash-4.2$ cat versions.txt

Red Hat 9
Fedora 38

Replacement successful.


-bash-4.2$ sed -i.bak 's/Red Hat/RHEL/g' versions.txt

-bash-4.2$ sed -i '/Fedora/d' versions.txt

--------------------------------------------------
Task 3: Text Processing with awk
--------------------------------------------------

-bash-4.2$ echo -e "ID,Name,OS\n1,Alice,RHEL\n2,Bob,Fedora" > users.csv

-bash-4.2$ cat users.csv

-bash-4.2$ awk -F ',' '{print $2}' users.csv
